# Flickr-data-extracter-v2

Simple Web Application to extract data from Flicker using Flickr API

link to web application : https://flickr-extractor.herokuapp.com/

Note: please select short date range(1 - 5 days), Web APP may crash if you select long date range. Please reload and select short range ( Reason - Heroku free verison doesn't allow enough resources to process large ranges)
